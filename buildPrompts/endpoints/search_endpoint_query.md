# Search Endpoint Query

GET /api/search

## Description

PURPOSE: Search for items with filters.

PARAMS:
- `q`: Search string
- `category`: string filter
- `minPrice`: number
- `maxPrice`: number
- `location`: GeoPoint (optional)

PROCESS:
1. Query Firestore `products` collection.
2. Apply text matching (via Algolia/ElasticSearch or simple prefix).
3. Apply range/category filters.
4. Return list of items.
