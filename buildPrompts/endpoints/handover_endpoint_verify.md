# Handover Endpoint Verify

## Description

PROCESS:
1. Validate token against `handover_transactions`.
2. Update `handover_transactions` status to 'completed'.
3. Update `rentals` status (e.g., 'active' or 'returned').
4. Trigger security deposit logic if 'return' is completed.
