# Auth Endpoint Google

POST /api/auth/google

## Description

PURPOSE: Authenticate user using Google ID Token.

PROCESS:
1. Verify Google token with Firebase Auth.
2. Check if user exists in `users` collection.
3. If new, create user record with `auth_provider: 'google'`.
4. Return user profile and session token.

SUCCESS (200): { user: UserObject, token: string }
