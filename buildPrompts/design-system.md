# Design System

This document outlines the visual and behavioral guidelines for the Near Share Flutter application.

## 1. Color Palette

Primary Theme: Material 3 (M3)
- **Primary**: #0017e4ff (Deep Blue) - Used for buttons, active states, and branding.
- **Secondary**: #fdc500 (Gold) - Used for accents and secondary information.
- **Surface**: #FFFBFE (White/Off-white) - Main background.
- **Error**: #B3261E (Red) - Validation errors and critical alerts.
- **Success**: #2E7D32 (Green) - Confirmation toasts and success states.

## 2. Typography

Font Family: `Inter Tight` (or default San Francisco/Roboto)
- **Display Large**: Size 32, Bold - For major headings.
- **Body Large**: Size 16, Regular - Standard text.
- **Label Small**: Size 11, Medium - Captions and small UI labels.

## 3. Component Patterns

- **Buttons**:
  - `FilledButton`: Primary actions (e.g., "Rent Now").
  - `OutlinedButton`: Secondary actions (e.g., "Message Owner").
  - `TextButton`: Minimal actions (e.g., "Cancel").
- **Cards**:
  - `ElevatedCard`: For item listings with subtle shadows.
  - `OutlinedCard`: For grouped settings or secondary info.
- **Feedback**:
  - `SnackBar`: For transient messages (errors/success).
  - `CircularProgressIndicator`: Native-style loaders.

## 4. Spacing & Grid

- **Base Unit**: 8px
- **Page Margin**: 16px (mobile standard)
- **Gutter**: 12px between grid items.
