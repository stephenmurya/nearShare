ABOUT THIS PROJECT
------------------

Near Share is a community-driven rental platform built with Flutter. It allows
users to list items for rent (cameras, event equipment, etc.) and others to rent
them locally.

Key Features:
- **Flutter Mobile App**: A cross-platform mobile experience for both owners and renters.
- **Search & Discovery**: Advanced filtering by category and location.
- **Risk Mitigation**: Automated security deposit collection and holding.
- **Handover Workflow**: Digital rental agreements and QR code verification for pick-up/return.
- **Admin Dashboard**: Web or in-app admin interface for platform management.
